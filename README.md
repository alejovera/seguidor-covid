### Características del Proyecto
Este proyecto intenta demostrar visualmente la propagación del COVID-19 en todos los países y en el mundo. Utiliza una base de datos abierta a la que se la hace un request para obtener la información. Esta base de datos nos devuelve un JSON que es manipulado para clasificar la información.
En este programa, además de todo el Frotend que se desarrollo, se manipula la información como sería si hubiese una base de datos en cualquier otro proyecto.
### Herramientas utilizadas
Se utilizó React JS y CSS principalmente. Además se utilizó Material UI para ciertos detalles visuales. Además se utilizaron librerías para el mapa y para formatear los números.

https://disease.sh/
